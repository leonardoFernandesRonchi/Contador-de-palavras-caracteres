# 🔢 Contador de Palavras e Caracteres

Aplicação simples e eficiente para contagem de palavras e caracteres em tempo real, ideal para redação, estudos e criação de conteúdo.

---

## 🚀 Funcionalidades

* 🔤 Contagem de palavras
* 🔡 Contagem de caracteres (com e sem espaços)
* 📏 Atualização em tempo real enquanto o usuário digita
* 📋 Suporte a texto colado
* ⚡ Interface rápida e leve

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---


## 📊 Como funciona

O sistema analisa o texto digitado ou colado e calcula automaticamente:

* Número de palavras (separadas por espaços)
* Número de caracteres totais
* Caracteres sem espaços

Esse tipo de ferramenta é muito usado para garantir limites de texto em redes sociais, redações e SEO ([Contador de Caracteres][1])
